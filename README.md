# JavonesYMazmorras
Este proyecto trata sobre un programa con diferentes clases, donde el usuario puede crear un personaje, dónde puede elegir de tres tipos de personajes diferentes: Mago, ladrón o guerrero, cada uno con sus respectivas características.

Después, también hay dos clases más aparte del tipo de personaje que uno quiere, y esas clases son las de Equipamiento y la de Inventario, dónde en el Equipamiento puede elegir el nombre y el tipo, si arma, armadura... Y en Inventario, sirve para añadir o ver los objetos disponibles que tiene el personaje.
